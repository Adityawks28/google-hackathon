FizzBuzz

Easy

Given an integer n, return a string array answer (1-indexed) where:

  • answer[i] == "FizzBuzz" if i is divisible by 3 and 5.
  • answer[i] == "Fizz" if i is divisible by 3.
  • answer[i] == "Buzz" if i is divisible by 5.
  • answer[i] == i (as a string) if none of the above conditions are true.

Example 1:

Input: n = 3

Output: ["1","2","Fizz"]

Great question! The order is crucial. Since "FizzBuzz" occurs when both conditions are met, you should check for i % 15 == 0 first.

If you check for 3 or 5 first, those conditions will trigger and the program won't even reach the combined check.
